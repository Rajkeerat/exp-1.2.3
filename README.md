# Experiment 1.2.3

## SVG Drawing Tool

### Aim
To build an interactive SVG drawing tool with mouse event handlers.

### Objectives
- Create an SVG canvas component  
- Implement mouse click event tracking  
- Draw shapes dynamically  
- Add undo functionality  
- Display number of shapes drawn  

### Tools Used
- HTML  
- CSS  
- JavaScript  
- SVG  

### Output
An interactive SVG canvas that allows users to draw and undo circles dynamically.
